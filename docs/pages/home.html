Clone the Repository

git clone https://github.com/E10H1M/rust-buck3t-ui.git

Getting Started

Create a .env file in the project root to configure the server:


BACKEND_BASE=http://127.0.0.1:8080
UI_HOST=127.0.0.1
UI_PORT=8085

These values are optional. If not set, the defaults are:

  • BACKEND_BASE: http://127.0.0.1:8080
  • UI_HOST: 127.0.0.1
  • UI_PORT: 8085

Running

cargo run

The UI will run on http://127.0.0.1:8085 by default, or whatever you set in your .env.
