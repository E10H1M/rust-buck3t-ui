Installing the requirements

Some requirements for Linux:

sudo apt update
sudo apt install pkg-config libssl-dev

Install Rust

Run the following command to install Rust via rustup:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Add Rust to your PATH (in the same terminal when installation finishes)

. "$HOME/.cargo/env"

Verify Installation

Restart your shell and run:

rustc --version
cargo --version
